# TropaCoinRAO

TropaCoinRAO is an open-source fintech project managed by Velatropa24Ventures CO (V24V), designed to beautify money through integrating 13:20 frequencies with principles of Sacred Economics and kin networks grouped into TropaNeur Tribes. This project, led by V24V's IT AlphaNode and fullstack developers, utilizes ICP Ninja's Tokenmania template to extend and build upon the foundation of a simple token minting application into a fully-fledged, production-ready Web3 dapp. Instead of decentralization, this project focuses on radialization, developing a social network dapp that functions as a Web3 rapp, a radialized application governed by a Radialized Autonomous Organization (RAO). Instead of DeFi, the TropaCoinRAO rapp functions as a RaFi, Radailized Finance technology, which authenticates users and accounts via Internet Identity. App users will use commons-backed capital to mint TropaCoin tokens (TPACO) on ICP. The owner principal of TPACO will be the user's Internet Identity principal.


## Phase 1 of TropaCoinRAO

* Apply Phase 0 workflow designs to create and mint TropaCoin tokens on ICP, backed by six types of capital.
* Deploy canisters, written in Motoko, to launch commons-backed TropaCoins onto ICP using the TropaCoinRAO ledger smart contract and its index smart contract.
* Use the ICRC-2 token standard to create and test TropaCoins as custom fungible tokens.
* Use the ICRC-37 token standard to create and test TropaCoin Tokens, itemized in Tropakals, as an NFT non-fungible token.
* Perform regular security reviews and audits to ensure robustness of the app, before and after testing

## Project structure

The `/backend` folder contains the Motoko canister, `app.mo`. The `/frontend` folder contains web assets for the TropaCoinRAO user interface. The user interface is written using Tailwind CSS and the React framework and the `mops.toml` file adds [Motoko dependencies](https://mops.one/) to the project.
